# Contact App (SQLite)
Roman Duda

Can't you remember all your contacts by yourself? 

Then this app might help you:
  
  *  Easily add contact's name and email
  *  Change the current contact
  *  Delete contacts

Tools implemented:

  * SQlite
  * Recycler view
  * Dialogue alert
  * Float Action button
